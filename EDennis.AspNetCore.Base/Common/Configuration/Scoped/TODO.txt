Middleware Pipeline
	1. DeveloperExceptionPage
    2. HttpsRedirection
	3. StaticFiles
	4. ApiLauncher
		- Launches one or more child APIs
    5. Routing	
	6. MockClient
		- *** ENVIRONMENT GUARD - NEVER IN PRODUCTION
			-- Adds a MockClient with requested scope for the current application,
	7. AdditionalClaimsMiddleware
		-- Adds Additional Claims to Claims Principal
	8. AuthenticationMiddleware
	9. AuthorizationMiddleware
	10. ScopePropertiesMiddleware
		- Stores the user name
		- Stores claims with designated patterns
		- Stores headers with designated patterns
	11. DbConnectionInterceptor
		- Only for apis with repos or dbcontexts
		- Replaces DbContextOptions in DbContextOptionsProvider to change databases/connections dynamically
	12. EndpointMiddleware
	13. SwaggerMiddleware
